Accessibility

Last updated 11 July 2026

We want as many people as possible to be able to use the Boudicca Productions website.

Using this website

We aim to provide clear headings, readable text, keyboard-accessible navigation, visible focus styles, meaningful link text and alternatives for informative images. The layout adapts to different screen sizes, and animations are reduced when your device requests reduced motion.

You should be able to zoom the page and use common browser or assistive-technology settings to adjust text and colour display.

Current limitations

The website is still in development and has not yet received an independent accessibility audit. Some placeholder production imagery and content will be replaced before launch. We will review accessibility again when real images, ticketing links and the mailing-list service are added.

Tell us about a problem

If you cannot access any part of this website or need information in another format, email hello@boudiccaproductions.com. Please tell us which page you were using and what went wrong. We will do our best to help and to improve the site.

Our approach

We use the Web Content Accessibility Guidelines (WCAG) 2.2 AA as a practical target. We do not claim full compliance until the finished site has been tested against that standard.
